This project is a creative C++ simulation that allows users to redesign different aspects of Disneyland. The program guides users through interactive choices involving attractions, park layout, and thematic elements. It demonstrates how programming can be used to model real‑world environments in a fun and engaging way while applying core C++ concepts such as functions, loops, conditionals, and structured program design.

Project Reflection
Summary of the Project and the Problem It Solved
The Redesigning Disneyland project allowed me to build an interactive C++ program that simulates redesigning parts of Disneyland. The problem it solved was organizing multiple user choices and design elements into a functional, menu‑driven program. It showed how programming logic can be used to model creative decision‑making and structure complex information in a clear, user‑friendly way.

What I Did Well
I did particularly well in organizing the program into readable sections and writing clear, well‑commented code. I made sure each function had a specific purpose, which helped keep the program structured and easy to follow. I also handled user input effectively, making the experience smooth and intuitive.

Where I Could Enhance My Code
I could improve the project by reducing repeated logic and making the program more modular. Adding more input validation would also make the program more secure and prevent unexpected behavior. These enhancements would make the code more efficient, scalable, and adaptable for future updates.

Most Challenging Parts and How I Overcame Them
The most challenging part was managing the different redesign options and keeping the program flow organized. I overcame this by breaking the project into smaller tasks, testing each part individually, and using debugging tools to track down issues. I also relied on documentation and course examples to guide my approach.
